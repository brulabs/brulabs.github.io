AutroPrime BS-200M

Das M steht für Marine

Versionen der Platinen

  • 1 = 2 USB Anschlüsse
  • 2 = 1 USB Anschluss (behebt Probleme mit Schleifentreibern)
  • 3 = blaue Platine (verbesserte Stabilität, bessere Integration der AutroGuard Melder)

Notstrombatterien Im Leerlauf sollte die Klemmenspannung einer vollen Batterie etwa 12,7V haben. Ist die Klemmenspannung nur noch 12,4V darunter kann man von einer Kapazität unter 60% ausgehen (Faustformel). Fällt die Kapazität beider Batterien unter 18V, schaltet das System ab. Dieser Zeitpunkt sollte erst nach frühestens 18 Stunden erreicht werden.

Servicemenü Level

  • Level 1 = Die ENTER Taste 3s lang gedrückt werden. Nur sehr eingeschränkte Ansicht.
  • Level 2 = Mit dem Schlüssel, wird für den Reset einer Störmeldung oder eines Brandalarm benötigt.
  • Level 3 = Das Servicemenü (alle Funktionen offen). Der Code ist 9999.

Anpassung der Bufferzeit bei Netzausfall Pfad: Service/Systemeinstellungen/Netzversorgung/Störanzeige/Verzögerte Störungsanzeige

Achtung

Nach Änderungen an Hard- und/oder Software immer die Konfiguration sichern und einen Neustart durchführen.

Softwareupdate Zwei USB-Sticks formatieren (FAT32), dann die Update-Dateien auf den einen USB Stick kopieren. Die aktuelle Konfiguration auf dem anderen USB-Stick sichern (siehe Export der aktuellen Konfiguration). Erst dann den USB-Stick mit den Update-Dateien einstecken. Update läuft automatisch durch.

Export der aktuellen Konfiguration USB Stick formatieren (FAT32) und formatierter USB-Stick in die Zentrale einstecken. Zu den Systemeinstellungen navigieren und dort Sichere Konfiguration anwählen. Unter Export und Import: Sicherung der aktuellen Konfiguration, dann Export Konfiguration (Config.xml).

Import der bearbeiteten Konfiguration USB-Stick mit bearbeiteter Konfiguration einstecken, die neue Konfiguration (Config.xml) importieren. Dann Wähle Konfiguration + Neustart System (Importierte Version wählen).

Autronica BX-10 (Obsolete)

Brandmeldezentrale, 4 Zonen (Schleifen), einschließlich Stromversorgung, ohne Batterien

Notstrombatterien Die BX-10 hat zwar eine Batterieüberwachung, allerdings beschränkt sich dieses auf die Spannungsüberwachung (niedrige Batteriespannung) und macht, soweit die Erkenntnis von M. Rohde, keinen Kapazitätstest wie die AutroPrime. Bei Batteriespannung unter 20,5 Volt soll es einen Störalarm geben.

Im Leerlauf sollte die Klemmenspannung einer vollen Batterie etwa 12,7V haben, Ist die Klemmenspannung nur noch 12,4V darunter kann man von einer Kapazität unter 60% ausgehen (Faustformel).

Wartung an der Zentrale Bedienerpasswort: 110 + ENTER Lampentest: 30 + ENTER

Brandmelder

Die AutroGuard-Serie ist die neuste Generation von Autronicas Brandmelder. Es gibt sie in den Versionen V-430 und V-530.

Ältere (obsolete) Brandmelder sind:

  • BD-26, BH-26, BD-27
  • BD-300, BH-300, BH-320 (inklusive SelfVertify)

AutroGuard V-430, V-530

Der Unterschied zwischen dem V-430 und dem V-530 ist, die V-530 gibt es als EX-geschützte Variante V-530-EXIA

Die Seriennummer Die ersten zwei Abschnitte einer Seriennummer 000.037.2023.058.00047 beschreiben den Typ, der dritte Abschnitt das Jahr, der vierte Abschnitt den Tag (1-365) und der letzte Abschnitt die eigentliche Seriennummer*

Operation Class MultiSensorWithHeat (MSWH) = Temperatur und Rauch (Standard) MultiSensorOnly (MSO) = Temperatur oder Rauch (Maschinenraum) HeatOnly (HO) = Temperatur

Der AutroGuard-Melder mit Operation Class MultiSensorWithHeat ist vergleichbar mit dem Vorgängermodell BHH-320, der AutroGuard mit Operation Class MultiSensorOnly ist vergleichbar mit den Vorgängermodellen BHH-300 und BHD-300 (zwei separate Melder in Maschinenräumen).

Achtung

Beim AutroGuard mit Operation Class MultiSensorOnly kann nur Rauch getestet werden!

Heat response categories

  • MultiSensorWithHeat = Heat response category A1 : ~54°C (Standard)
  • MultiSensorOnly : Heat response category B : ~69°C (z.B. Maschinenraum)

Wissenswertes

  • Neu ist bei den Meldern die Adresse im Sockel.
  • AutroGuard Retro ersetzt Melder der Serie 200/300/500. Identisch AutroGuard, aber mit anderem Sockel.
  • Die AutroGuard Melder haben SelfVertify 2.0
  • Das -M steht für die Marineveriante
  • Die Filterkonstante beim Wärmemelder beträgt 15-30 Sekunden

Brandmelder ersetzen

Defekte Brandmelder können jeweils durch einen identischen Melder getauscht werden. Wurde der Melder ersetzt, die Meldung "Schleifengerät wurde getauscht" im Display der Zentrale abwarten und dann einen Reset an der Anlage durchführen. Wurde der Melder gegen einen anderen Typ getauscht, die Meldung "Schleifengerät ist ersetzt worden" abwarten und dann die Konfiguration sichern und einen Neustart durchführen. Dies gilt für alle Melder-Typen.

Achtung

Nach Änderungen an Hard- und/oder Software immer die Konfiguration sichern und einen Neustart durchführen.

Brandmelder prüfen

Die Brandmelder werden mit dem testifire-100 oder, falls dies nicht möglich ist, mit Prüfgas und einem Heissluftfön getestet. Da bei den Limmatbooten die Melder nicht zugänglich sind, wird dort ein SelfVerify Test durchgeführt.

AutroClient Software

Um die Anlage zu konfigurieren, braucht es die AutroClient Software. Via URL oder Open Log ... kann eine Verbindung zur Anlage gemacht werden oder die Konfigurationsdatei geöffnet und bearbeitet werden.

Monitoring Standard, ist reine Ansicht. Keine Anpassungen machbar.

Control Es können keine Änderungen gemacht werden, sondern Konfigurationen. PW = 1234

Configuration Hier werden die Änderungen gemacht. PW = 9999

ÜZ (Überwachungszone) englisch DZ Neue Gruppen (Zonen) anlegen, die entsprechenden Melder zuordnen

AZ (Alarmzone) Nichts verändern, bleibt Standard.

CauseEffect Unter Inputs neu erstellte Gruppen (Zonen) hinzufügen (per Drag & Drop von rechts nach links ziehen). Falls vergessen kommt Fehlermeldung ÜZ aktiviert keine AZ Unter Outputs bleibt alles Standard.

Einstellung Operation Class MultiSensorWithHeat = Temperatur und Rauch MultiSensorOnly = Temperatur und/oder Rauch HeatOnly = Nur Temparatur

Achtung

Immer beide, Operation class day und night, anpassen!

Performance day class Messzeit, Wert nicht verändern.

Begriffe Topologie = Reihenfolge Punkte = Melder ÜZ aktiviert keine AZ ÜZ = DZ

AS-2000 Schleifenmessgerät

Mit dem Schleifenmessgerät WAS-2000 können die Schleifen ausgelesen werden, zwecks Störungssuche oder um die ausgelesenen Daten als Datei abzuspeichern. Dazu wird die Software AS-2000 benötigt.

WAS-2000 Schleifenprüfgerät Prüfkoffer WAS-2000 Interface mit Software AS-2000

Inbetriebnahme

WAS-2000 via USB mit PC verbinden. Dann die zu messende Schleife mit in der Zentrale abhängen und mit den Prüfklemmen des WAS-2000 verbinden. Batterien verbinden und Netzstecker mit Steckdose verbinden.

Software AS-2000 starten und den entsprechenden COM-Anschluss wählen (ausprobieren bis es klappt).

Die verschiedenen Möglichkeiten sind in der Dokumentation beschrieben.

SelfVerify Test

Drähte der entsprechenden Schleife von der Zentrale abhängen und an den WAS-2000 anschliessen. Im Programm AS-2000 den START Knopf drücken, die Analyse startet. Wenn die Analyse fertig ist, Report drücken und abspeichern. Dann SV test drücken und starten. Der Test (pro Melder) läuft, wenn beendet, ebenfalls speichern.

Dies muss für jede Schleife gemacht werden, es gibt pro Schleife jeweils 2 Dokumente.

Bei beiden Dokumente folgendes nach der Zeile "Serviced by" einfügen (und anpassen: Batterieräume Bb, Stb, Maschinenraum):

:::::::::::::::::::::::::::::::::::
Zürichsee Schifffahrtsgesellschaft
Schiff : eLimmatboot Pontus
Prüfung : Batterieräume Stb
:::::::::::::::::::::::::::::::::::

Das Dokument Report wird Portrait ausgedruckt. Das Dokument SV-Test muss vorher auf Landscape gedreht werden. Dann ausdrucken. Siehe Beispiele:

  • Loop Report
  • SelfVertify Test Result
